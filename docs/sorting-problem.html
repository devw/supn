Exercises 2

Given a list of positive numbers, write a python function to find the difference between the maximum and the minimum of an array without using inbuilt functions min, max, sort and sorted.. 


def bubbleSort(arr):
    for i in range(len(arr)):
        for j in range(len(arr) - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr


def getMinMaxDelta(arr):
    return arr[-1] - arr[0]


print(f"getMinMaxDelta {getMinMaxDelta(bubbleSort([10, 3, 4, 5]))}")
Exercises 3

Find Closest Numbers

Sorting is useful as the first step in many different tasks. The most common task is to make finding things easier, but there are other uses as well. In this case, it will make it easier to determine which pair or pairs of elements have the smallest absolute difference between them.

Input_1: [2, 5, 4, 6, 7]

Output_1: [(4, 5), (5, 6), (6, 7)]

Input_2: [4, 10, 6, 8, 9]

Output_2: [(8, 9), (9, 10)]

Input_3: [4, 10, 7, 8, 9, 10]

Output_3: [(10, 10)]

Input_4: [-4, 10, -3, 8, 9, 12]

Output_4: [(-4, -3), (8, 9), (9, 10)]


def closestNumbers(arr):
    dictionary = {}
    arr = sorted(arr)

    for ind in range(1, len(arr)):
        diff = abs(arr[ind-1] - arr[ind])
        dictionary[(arr[ind-1], arr[ind])] = diff
    minimum = min(dictionary.values())

    return[k for k, v in dictionary.items() if v == minimum]
