Exercise 1

write a function to return the number of occurrences (or frequency) in a array.

example_1: IN: [0, 1, 1, 1, 2], 2; OUT: [1, 3, 1]

example_2: IN: [1, 1, 1, 2, 4], 4; OUT: [0, 3, 1, 0, 1]


def get_occurrences(nums, n):
    """
    write a function to return the number of occurrences (or frequency) in an array.
    example: 
    IN: [2, 1, 2, 2], OUT: [0, 1, 3]
    """
    occurrences = [0] * (n + 1)
    for num in nums:
        occurrences[num] += 1
    return occurrences


nums = [1, 1, 1, 1, 1, 1, 1, 2, 4]
res = get_occurrences(nums, 4)
print(res)

Exercise 2

Write a function to sort a list of numbers, whose elements are less than or equal to n - 1, using the list of occurrences given by the occurrences function.

example_1: IN: [0, 2, 1], 2; OUT: [0, 1, 2]

example_2: IN: [1, 1, 4, 3, 0], 4; OUT: [0, 1, 1, 3, 4]


def counting_sort(nums, n):
    """Write a function to sort a list of numbers, 
    whose elements are less than or equal to n - 1, 
    using the list of occurrences given by the occurrences function"""
    sorted_nums = []
    occurrences = get_occurrences(nums, n)
    for k in range(n):
        for i in range(occurrences[k]):
            sorted_nums.append(k)
    return sorted_nums


nums = [1, 1, 2, 4, 1, 0]
res = counting_sort(nums, 5)
print(res)
